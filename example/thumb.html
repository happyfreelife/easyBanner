自定义样式 函数和方法 缩略图 延迟加载 移动端 
/**
 * 同时使用参数thumbWidth, thumbHeight
 * 即可开启缩略图功能
 * 缩略图会替换原来的指示按钮
 */

$thumb.terseBanner({
	thumbWidth: 120,
	thumbHeight: 80
});

/**
 * 不指定图片对应的缩略图，缩略图就是原图片
 * 指定图片对应的缩略图：在img的src属性值后面添加 ?thumb=... (...为缩略图的路径)
 * <img src="img/thumb-1.jpg?thumb=img/thumb-small-1.jpg">
 */

/* 可以使用自定义样式来修改缩略图 */
.thumb .tb-btn a{
	margin: 0 10px;
	border: 1px solid #fff;
	border-radius: 0;
	opacity: .5;
}
.thumb .tb-btn a.active{
	opacity: 1;
}
